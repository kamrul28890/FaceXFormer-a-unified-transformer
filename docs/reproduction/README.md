# FaceXFormer Reproduction Documentation

This directory is the human-readable documentation hub for the reproduction work in this repository.

## Documents

- [Project Overview](PROJECT_OVERVIEW.md) summarizes the goal, scope, repository layout, final report package, and current status.
- [Implementation Details](IMPLEMENTATION_DETAILS.md) documents the rebuilt model, data, training, evaluation, and reporting pipeline.
- [Artifact Manifest](ARTIFACT_MANIFEST.md) lists the reports, generated figures, result folders, checkpoints, scripts, and source materials that are part of the delivered reproduction package.

## Final Report

The polished report package is stored separately at:

- `report_final/main.pdf`
- `report_final/main.tex`
- `report_final/assets/`
- `report_final/draft_sources/`
- `report_final/source_materials/`
- `report_final/rendered_pages/`

The final report intentionally omits ablation studies because the project did not complete validated ablation experiments.
