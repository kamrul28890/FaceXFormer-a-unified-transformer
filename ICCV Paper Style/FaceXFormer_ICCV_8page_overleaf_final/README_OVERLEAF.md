# FaceXFormer ICCV Overleaf Package

Upload the contents of this folder to Overleaf and set `main.tex` as the main file.

Included files:
- ICCV style files: `iccv.sty`, `ieeenat_fullname.bst`
- Paper source: `main.tex`, `preamble.tex`, `sec/report.tex`
- Bibliography: `main.bib` plus compiled `main.bbl`
- Figures: `assets/`
- Verified local output: `main.pdf`

The package was prepared from the local ICCV author kit and should compile with pdfLaTeX on Overleaf.
