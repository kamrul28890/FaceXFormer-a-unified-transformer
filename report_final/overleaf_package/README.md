# FaceXFormer Overleaf Package

This folder is ready to upload to Overleaf as a zip project.

## Compile Settings

- Main file: `main.tex`
- Compiler: pdfLaTeX
- TeX Live version: current Overleaf default is fine

The report uses only standard LaTeX packages and local assets in `assets/`.

## Contents

- `main.tex` - final report source
- `assets/` - all figures required by `main.tex`
- `.latexmkrc` - tells Overleaf/local latexmk to compile with pdfLaTeX
