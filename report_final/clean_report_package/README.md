# FaceXFormer Report Package

This folder contains the clean report deliverables and all assets needed to compile the LaTeX version.

## Main files

- `FaceXFormer_Report_v2_clean.tex` - clean TeXmaker-ready LaTeX source.
- `FaceXFormer_Report_v2_clean.pdf` - rendered clean professional PDF.
- `FaceXFormer_Report_v2.docx` - Word version generated from the v2 report source.
- `FaceXFormer_Report_v2.pdf` - PDF rendered from the Word version.
- `assets/` - graph assets referenced by the LaTeX source.

## Compile

Open `FaceXFormer_Report_v2_clean.tex` in TeXmaker and run PDFLaTeX twice. The file expects the `assets/` folder to remain beside it.
